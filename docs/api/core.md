# @autoguide/core

Framework-agnostic types, fact model, knowledge graph, review queue, plugins, and scan helpers.
No React/DOM imports — safe for Node CLI and browser bundles.

## Key public APIs

| API | Description |
|-----|-------------|
| `Fact` | Provenance-backed knowledge atom (`types/fact`) |
| `KnowledgeGraph` | Merge and conflict-detect facts (`graph/knowledge-graph`) |
| `ReviewQueue` | Manual review queue with override precedence (`review/review-queue`) |
| `generateRecommendations` | Deterministic dev hints from scan hints and facts |
| `PluginRegistry` | Register scanner/adapter plugins with lifecycle hooks |
| `runPluginScans` | Isolated plugin scan phase (non-fatal failures) |
| `generateToursFromFlows` | Build tours.json from flow records |
| `validateScanArtifacts` | Runtime type guards for scan JSON artifacts |
| `configureRedaction` | PII/secret redaction before persistence |
